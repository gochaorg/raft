Справка о программе
=========================

    > log-tools { опции } комманда { комманда }

Опции

- `+v` - вывести подробную информацию о выполнении
- `-v` - не выводить подробную информацию

Команды

- `a` либо `append` - добавление записи/файла в лог
- `v` либо `view` - просмотр заголовков лог файла
